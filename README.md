# Color Clock

## Try it locally

1. Run:
    ```bash
    $ git clone https://github.com/MartinRosenberg/Color-Clock.git
    $ cd Color-Clock
    $ yarn
    $ yarn start:dev
    ```
2. Navigate to http://localhost:8080/
