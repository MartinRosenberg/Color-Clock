# React-Hex-Clock

## Try it locally

1. Run:
    ```bash
    $ git clone https://github.com/MartinRosenberg/React-Hex-Clock.git
    $ cd React-Hex-Clock
    $ yarn
    $ yarn start:dev
    ```
2. Navigate to http://localhost:8080/
